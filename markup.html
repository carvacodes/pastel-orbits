Hover/Tap for controls

  • In case you want to use a keyboard to control the animation:

  • Spacebar: reset
  • X-Y: start/stop auto X and Y scaling animation
  • U-D Arrows: change the orbit size
  • L-R Arrows: change the number of dots
  • W-S: increase/decrease orbit speed
  • E-D: adjust the X-axis scale
  • R-F: adjust the Y-axis scale
  • Q-A: adjust dot size
